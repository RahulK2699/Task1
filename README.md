<ui>

# How to use
Clone repo into your local machine
Move to /ass1 folder to your terminal follow  steps as given below
<br>
  <ui>
    <li>npm install</li>
    <li>json-server -port 3001 --watch db.json [to run JSON-SERVER]</li>
    <li>npm start</li>
  </ui>
